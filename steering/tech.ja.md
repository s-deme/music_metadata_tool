## Overview

Python 製のクロスプラットフォーム CLI で、音楽ファイルのメタデータを読み取り・検証・一括更新・整形する。依存は最小限とし、CSV/TSV 往復と再現性の高いバッチ処理を重視する。

### Python Ecosystem

- **CLI**: Typer + Rich（ヘルプ/出力整形）

## Tooling & Workflows

- **Type Check**: `mypy src`
- **Test**: `pytest`

## Platform Notes

- 対応 OS: Windows / macOS / Linux
- パスは pathlib を使用し、改行は OS デフォルト。CSV/TSV は UTF-8（BOM なし）で出力。
- エンコーディング: UTF-8 を基本とし、非 UTF-8 で読み込んだ場合は警告。Shift-JIS などレガシーは validate で報告。
- 画像処理は当面 Pillow 非採用。カバーアートはバイナリ差し替えに留め、リサイズ等は将来オプションで検討。
- パフォーマンス目標: 1,000 ファイルを 60 秒以内、メモリ 400MB 以内。大規模は逐次/チャンク処理で対応。
